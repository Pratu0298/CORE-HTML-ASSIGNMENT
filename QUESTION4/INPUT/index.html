List of tags

  1. html : HTML tag help to convert the html document into web browser
  2. form : The form tag is use to create form for user input.
  3. img : The img tag is used to embed an image in an HTML page.
  4. a : anchor tag or a tag defines a hyperlink, which is used to link from one page to another.
  5. hr : It defines the horizontal line.
  6. h1 : It defines the heading of level one.
  7. p : It defines the paragraph.
  8. br : It breaks the line.
  9. div : It defines the container for HTML element.
  10. table : It defines the table for HTML page.
  11. audio : The tag is used to embed sound content in a document.
  12. body : body tag defines the document of body.
  13. span : The span tag is an inline container used to mark up a part of a text, or a part of a document.
  14. video : The video tag is used to embed video content in a document.
  15. ul : It defines the unordered list.
  16. ol : It defines the ordered list.
